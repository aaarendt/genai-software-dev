What is a harness?

A harness is the surrounding system — the file access, tool permissions, and execution environment wrapped around the model. Saying "I'm using Claude" is under-specified: browser Claude, Claude Code, and Claude in VS Code all share the same underlying model but have completely different harnesses. The harness is what has changed most in the last two years, and it determines what AI can actually do on your behalf.
